# 橙鸭笔记系统_笔记图片
存储了笔记中插入的图片以及图片扫描文本, 可能有少部分其他文件.  
命名规则: ``<上传时间>_<图片序号>_<六位随机数>.<文件名最后一个"."后面的文本(后缀名)>``
（其中开头带有"replenish_"的是后期上传文件.）  
例子: 1661441149339_1_279154.jpeg  
旧版命名规则: ``<上传时间>_<六位随机数>.<文件名最后一个"."后面的文本(后缀名)>``  
例子: 1659455173612_518183.png
